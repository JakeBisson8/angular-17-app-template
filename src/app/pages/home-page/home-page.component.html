Angular 17 Starter Template

About

This is a very basic Angular 17 starter template that comes pre-configured with TailwindCSS, Angular Material, ESLint, Prettier and Husky

Getting Started

As this is a very basic starter template, you are free to do what you please from this point onwards. The template comes with a simple folder structure that you may choose to use or modify based on your project preferences. The following is the structure and the intended purpose of each folder (but you probably could've guessed what they would store already):

  • components/: holds all re-usable components
  • models/: holds all typescript models
  • pages/: holds all components representing pages
  • services/: holds all services

You may clear this template your are currently viewing in ./src/pages/home-page/home-page.component.html file to get started building your application's home page.

npm run start to start the development server.
